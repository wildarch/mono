# Structure
Top-level:
- Declaration
- Function definition

## Declaration
https://en.cppreference.com/c/language/declarations
Two parts:
- specifiers and qualifiers (typedef, const)
- declarators and initializers (identifier, value)

specifiers:
- type
- storage class (typedef, static, extern)
- function specifier (inline)

typedef cannot be prefixed!

key kinds of declarations:
- typedef
- variable
- function
- enum
- struct
- union
